The main difference between CSS Grid Layout and CSS Flexbox Layout is their existed dimansion. Grid is made for two-dimensional layout and Flexbox is for one. This means Flexbox can work on either row or columns-only one at a time, but Grids can work on both in the same time.

Bootstrap is a front-end framework to develop websites. It offers pre-built elements such as navigation, typography, buttons, and forms. The templates provided by Bootstrap are designed using either JavaScript or CSS (Cascading Style Sheets).On the other hand Tailwind is also a CSS framework responsible for creating a website. The styling process in Tailwind CSS is speedy than other front-end frameworks.

In web development, the CSS box model refers to how HTML elements are modeled in browser engines and how the dimensions of those HTML elements are derived from CSS properties. It is a fundamental concept for the composition of HTML webpages.

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.
